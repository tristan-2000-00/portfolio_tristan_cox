Dash & Dine

Take your cooking to the next level with our all-in-one responsive recipe web app with a vast collection of recipes ranging from quick and easy weeknight dinners to gourmet dishes!

Project Goal

To create a web app with a large variety of recipes that can be found with ease and precision, that will result in an overall less frustrating and more enjoyable experience.

This web app will allow users to find search and filter recipes, post their own recipes, and interact with other users.

User Personas

After extensive market research and valuable user insight, I created user personas to help create an in-depth analysis of potential users.

MVP Document

To define the requirements of my web app, I created an MVP which were supported by user stories.

Requirements
  • Allow users to create an account
  • Allow users to search for recipes and filter the results
  • Allow users to save recipes in one place
  • Allow users to interact with other users

Initial Problems

Key Insights

The web app will need to add a bottom navigation bar to the mobile and tablet breakpoints

The web app should have a simpler filtering system

The web app should implement more back arrow sand easily identifiable navigational icons
